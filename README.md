# Vilm React

## Guide Instalation

### Requirments

1. Node.js >= 18
2. Typescript
3. PNPM

### Steps

1. Clone the repository

```bash
git clone https://github.com/nnivxix/vilm.git
```

```bash
cd vilm
```

```bash
pnpm install
```

2. Copy the `.env` file

```bash
cp .env.example .env
```

3. Get the API KEY

Go to .env file and fill the value `TMDB_API_TOKEN=` with key from [TMDB API](https://developer.themoviedb.org/docs/getting-started)

4. Enjoy

Happy Coding
